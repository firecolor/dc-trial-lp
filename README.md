## ファイル構成
```
lp
├─ _complete : 完成したファイル
│    ├─ assets
│    └─ index.html
│
├─ _your-space : 作業する場所（フォルダ名を変えて使用します）
│    ├─ assets : アセットフォルダ。制作に必要なファイルを保存
│    └─ index.html：これを編集して学習していきます
│
├─ README.md : このファイル

```
